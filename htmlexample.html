HTML Example 1: Basic Structure

<h1>Hello World</h1>
<p>This is my first HTML example.</p>

This is my first HTML example.

HTML Example 3: Semantic Tags

       <article>
        <h2>What is HTML?</h2>
        <p>HTML stands for HyperText Markup Language.</p>
       </article>
Output:

What is HTML?

HTML stands for HyperText Markup Language.

HTML Example 5: Table Tag

<table border="1">
<tr>
  <th>Name</th>
  <th>Age</th>
</tr>
<tr>
  <td>Alice</td>
  <td>24</td>
</tr>
<tr>
  <td>Bob</td>
  <td>28</td>
</tr>
</table>
Output:
Name Age
Alice 24
Bob 28
