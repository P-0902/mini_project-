Introduction to HTML

HTML (HyperText Markup Language) is the foundation of web development. It is used to structure content on the web and is the first step in creating a webpage. HTML uses tags to define elements like headings, paragraphs, links, images, and more.

Basic HTML Elements

HTML elements are the building blocks of HTML. Here are some examples:

Headings

<h1>This is Heading 1</h1>
<h2>This is Heading 2</h2>
<h3>This is Heading 3</h3>

Paragraphs

<p>This is a paragraph.</p>

Links

<a href="https://www.example.com">Click Here</a>

Images

<img src="https://via.placeholder.com/300" alt="Placeholder Image">

HTML Attributes

Attributes provide additional information about HTML elements. Examples:

<a href="https://www.example.com" target="_blank">Open in New Tab</a>
<img src="image.jpg" alt="Description" width="500" height="300">

HTML Lists

Ordered List

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

Unordered List

<ul>
    <li>First item</li>
    <li>Second item</li>
</ul>

HTML Tables

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>30</td>
    </tr>
</table>

HTML Semantic Elements

These elements provide meaning to the structure of a webpage:

<header>Header content</header>
<nav>Navigation links</nav>
<article>Main article content</article>
<footer>Footer content</footer>

HTML Forms

<form>
    <label>Name: <input type="text"></label>
    <input type="submit" value="Submit">
</form>

HTML Special Characters

<p>Use &amp; for &, &lt; for <, and &gt; for >.</p>

Conclusion

HTML is essential for web development. By mastering its elements, attributes, and structures, you can create powerful web pages. Keep practicing to enhance your skills!
