CSS Example 4: Hover Effect

/* CSS */
.hover-effect {
    display: inline-block;
    padding: 10px 20px;
    background-color: #2196f3;
    color: white;
    border-radius: 5px;
    text-decoration: none;
}

.hover-effect:hover {
    background-color: #1976d2;
    transform: scale(1.1);
    transition: all 0.3s ease-in-out;
}
Output:
Hover Me!

CSS Example 9: Responsive Design (Media Query)

  /* CSS */
  .responsive-box {
      width: 100%;
      max-width: 300px;
      background-color: #4caf50;
      padding: 20px;
      text-align: center;
      margin: 20px auto;
  }
  
  @media (max-width: 600px) {
      .responsive-box {
          background-color: #ff5722;
          color: white;
      }
  }
Output:
Resize the browser window to see changes!

CSS Example 10: Random Bar Animation

/* CSS */
.random-bar {
    width: 100%;
    height: 30px;
    background-color: #4caf50;
    margin: 20px auto;
    animation: randomWidthColorChange 5s linear infinite;
}

@keyframes randomWidthColorChange {
    0% { width: 50%; background-color: #FF5722; }
    25% { width: 75%; background-color: #2196F3; }
    50% { width: 100%; background-color: #4CAF50; }
    75% { width: 25%; background-color: #FFEB3B; }
    100% { width: 50%; background-color: #FF5722; }
}
