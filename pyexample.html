Python Example 7: Dictionary Manipulation

# Python Code

student["age"] = 22
student["grade"] = "A"
print(student)
Output:

{'name': 'John', 'age': 22, 'course': 'CS', 'grade': 'A'}

Python Example 8: String Formatting

Python Example 9: File Handling

# Python Code
with open("example.txt", "w") as file:
  file.write("This is a test file.")
  
with open("example.txt", "r") as file:
  content = file.read()
  print(content)
Output:

This is a test file.

Python Example 10: Class and Object Basics

# Python Code
class Person:
  def __init__(self, name, age):
      self.name = name
      self.age = age

  def introduce(self):
      return f"My name is {self.name} and I am {self.age} years old."

person1 = Person("Bob", 30)
print(person1.introduce())
Output:

My name is Bob and I am 30 years old.
